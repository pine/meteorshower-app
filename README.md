## Requirements

- Yarn
- Node `$(cat .node-version)`

## Getting started

```
$ git clone https://github.com/meteor-sw/MeteorShower.git
$ cd MeteorShower

$ yarn
$ yarn start     # for development
$ yarn run build # for production
```

## Test

```
$ yarn test
```
